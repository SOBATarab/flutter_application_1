# Stitch Google AI Plugin

Local Codex plugin for routing Google Stitch UI design requests into a repeatable workflow.

This plugin currently provides a Codex skill and presentation metadata. It does not install or authenticate the Stitch MCP server automatically. To enable MCP-backed Stitch tools, follow Google's current Stitch MCP setup docs:

https://stitch.withgoogle.com/docs/mcp/setup
